Project Work

Dynamic Sine Wave, Jan 2019

It's a sine wave if each of its waves were uniquely and randomly generated...

If you had a value x that went up and down between the value of 1 and -1, then you plotted that value on a 2D graph, you would have a sine wave. I wanted to know what a sine wave would look like if you made every wave segment randomly generated. Here's how it turned out!

Rotational Solids, Mar 2018

The Rotational Solids graph is one of my favorite projects. When I decided that I wanted to get serious about programming for myself, this is what I created. In order to understand what exactly you're looking at you need to know a little bit about the background. In Calculus 2 I learned that you could measure the volume of a water tower by using integrals. When a 2D curve is rotated it projects into 3D space, and it can approximately capture the shape of a 3D object.

That's exactly what you're looking at! I craft a 3D graph, plug in a function like x^2 , tell it to rotate, and there's my rotational solid!
